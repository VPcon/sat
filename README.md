# VP Consulting — портфолио

Сайт-портфолио для презентации работ клиентам: веб-приложения, интернет-магазины, лендинги и сайты-визитки.

## Как открыть

Просто открой `index.html` в браузере — никакой сборки или сервера не нужно, всё работает локально.

Чтобы отправить клиенту: можно загрузить папку на любой хостинг (Vercel, Netlify, GitHub Pages — бесплатно и за 2 минуты) или просто прислать ссылку, если уже на хостинге.

## Структура

```
VP_Consulting_Portfolio/
├── index.html         — главная страница портфолио
├── styles.css         — дизайн-система
├── app.js             — анимации при скролле, lightbox с деталями кейсов
├── assets/            — скриншоты кейсов
└── mockups/           — полные интерактивные версии трёх макетов
    ├── cafe-landing.html   — лендинг кофейни «Зерно»
    ├── shop.html            — интернет-магазин ShopRun
    └── lawyer.html          — сайт-визитка «Громов & партнёры»
```

## Что показано

- **Кейс TaskFlow AI** — ссылается на десктоп-приложение, которое мы сделали ранее (используются те же скриншоты)
- **3 концепт-макета** — кликабельны, открывают живую версию страницы в новой вкладке через lightbox
- **Bento-grid услуг** — 4 направления работы
- **Процесс из 5 шагов** — что обычно спрашивают клиенты до старта
- **Форма контакта** — сейчас просто `mailto:` ссылка, замени на свою почту/Telegram в `index.html` (секция `#contact`)

## Что стоит поменять перед отправкой клиентам

1. В `index.html` в самом низу — замени `hello@vpconsulting.dev` на свою настоящую почту
2. Ссылку `Telegram` (там сейчас `href="#"`) — замени на свой профиль
3. По желанию — замени скриншоты в `assets/` на скриншоты твоих реальных будущих проектов, когда они появятся
